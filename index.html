FoundationKit by foundationkit

Everything that really should be in Foundation, but isn't.

Still very much work-in-progress. It's partly based on PSFoundation (https://github.com/steipete/PSFoundation)

Install

git clone https://foundationkit@github.com/foundationkit/FoundationKit.git git submodule init && git submodule update
